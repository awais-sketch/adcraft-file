02 / How we think about it

Compliance is not a constraint — it is how it's done well.

We write copy that is already inside the lines — because that is the only way to write healthcare copy well.

01

Compliant from the first line

We already know which words trigger compliance review — testimonials, comparatives, claims that imply a guaranteed outcome. So we never write them in the first place, instead of rewriting an entire campaign after the fact.

02

Built for how patients decide

Persuasive without ever crossing AHPRA. Patients choose a practice on trust, proximity and clarity — so the copy leads with those, and earns the booking without a single claim that puts the practice at risk.

03

Performance follows

Compliant copy, written for the real decision a patient is making, simply performs better. Honesty reads as confidence — and confidence is what converts. The guidelines and the results point the same way.
